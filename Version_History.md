# Version History

This is the full version history with all the release notes previous updates.

## Version 1.0.0 - Hello World

This is the first iteration of the script and very basic, it picks out the owners of the given project-ID.

## Version 1.1.0 - Save File

The feature to save the requested owners in a dedicated txt-file is now active.

## Version 1.1.1 - Stability Increase

This is a small update with:

- More robust error management
- Error codes
- Better role inheritance management (filter Service Agents)
- A bit lighter code thanks to using more variables

## Version 1.1.2 - CSV has arrived

This is one of the biggest updates yet!

- Users can now save the owners in a dedicated directory both in raw data as a CSV-file and a TXT-file.
- Even more robust upgrades to ensure that the project exists and handle incorrect input better.
- Output messages now appear with less clutter in the terminal.
- Brand new example files.
